# grpc-java-starter

Simple starter for grpc-java based projects.

### Requirements

- [Gradle](https://gradle.org/releases/) 8.2.1
- [JDK 17](https://www.azul.com/downloads/?package=jdk#zulu) from Azul

### Build

```bash
./gradlew clean build
```

### Run

Server
```bash
./gradlew :server:run
```

Client
```bash
./gradlew :client:run
```
